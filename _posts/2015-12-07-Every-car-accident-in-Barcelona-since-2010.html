Every Barcelona car accident from 2010-2014

Zoom in and around, and click on the markers for details about each accident.

Data sources, code, etc.

OpenDataBCN hosts the dataset for traffic accidents managed by the Guardia Urbana.

This map was generated in R, using the leaflet library. The code and data for this post (including source files, cleaning script, visualizations, etc.) can all be found on github.
